# Operating Procedure Standardisation with VBA-Powered User Forms in Microsoft Excel

VBA-powered user forms in Excel enable fast development and deployment of spreadsheet applications that can perform a variety of tasks that standard Excel features cannot achieve. Using drag-and-drop, developers can create simple but effective UIs and add VBA programs to achieve whatever they want, whether it is a financial model, a checklist, or something else.

Excel remains the most popular spreadsheet software in the market. VBA will stay relevant as long as Excel is used across industries and geographies. In this project, I created an Excel-based spreadsheet application with user forms to standardise the operating procedure of the credit services department of a global financial institution. This application is also used for training new employees joining the department as it helps flatten the learning curve.
